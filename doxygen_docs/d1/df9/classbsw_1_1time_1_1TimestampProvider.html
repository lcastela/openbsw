uint32_t bsw::time::TimestampProvider::getTimestampMs32Bit ( )
static

Provides a 32 bit CPU local and relative timestamp in milli seconds resolution.

Will overflow after about 49 days. Therefore only suitable for medium term measurements.

A 32 bit timestamp is less expensive compared to a 64 bit timestamp, at least on 32 bit platforms. Therefore prefer 32 bit timestamp if feasible.

Returns
32 bit CPU local timestamp milli seconds.

uint64_t bsw::time::TimestampProvider::getTimestampMs64Bit ( )
static

Provides a 64 bit CPU local and relative timestamp in milli seconds resolution.

The 64 bit timestamp value is guaranteed to not wrap around before 2^64, i.e. using the whole 64 bits for counting.

Returns
64 bit CPU local timestamp milli seconds.
