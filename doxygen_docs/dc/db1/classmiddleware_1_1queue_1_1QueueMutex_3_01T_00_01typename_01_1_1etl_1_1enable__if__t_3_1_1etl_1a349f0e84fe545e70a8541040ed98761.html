Detailed Description

template<typename T>
class middleware::queue::QueueMutex< T, typename ::etl::enable_if_t<::etl::is_pointer< T >::value > >

Specialization for pointer to integer mutexes. In this specialization, when creating the mutex_t alias we first remove the pointer and then add the volatile to T and finally add the pointer again. This is because (volatile uint8_t*) != (uint8_t* volatile) and we want the first one.
