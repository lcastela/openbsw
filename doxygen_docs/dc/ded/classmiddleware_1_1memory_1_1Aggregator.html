Detailed Description

template<typename... T>
class middleware::memory::Aggregator< T >

Memory aggregator that aggregates multiple memory pools of different chunk sizes.

Template Parameters
TTypes of memory pools to aggregate.

Constructor & Destructor Documentation

◆ Aggregator()

template<typename... T>
middleware::memory::Aggregator< T >::Aggregator ( uint8_t volatile *const  pLockElement)
inlineexplicit

Constructor which initializes the aggregator.

This ensures pools are always initialized when an Aggregator is constructed, eliminating the risk of using an uninitialized instance in multicore systems, construction must complete on one core before others access the aggregator. The external lock element and all pools must be fully initialized before any allocation.

Parameters
pLockElementpointer to the external lock element used by the base allocator
