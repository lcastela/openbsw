Detailed Description

template<uint8_t MAX_NUM_CHANNELS>
class routing::Router< MAX_NUM_CHANNELS >

A PDU Router that forwards PDUs from readers to writers according to its routing table.

[TPARAMS_BEGIN]

Template Parameters
MAX_NUM_CHANNELSThe maximum number of channels to write to and from. Needs to fit the routing table data. [TPARAMS_END]

Initialize a Router by loading the PduRoutingTable from the provided config byte array.

If the config data is invalid, the Router will be uninitialized and the other functions will not do anything apart from logging an error.

Otherwise, it starts with transmission and reception enabled for all channels.
