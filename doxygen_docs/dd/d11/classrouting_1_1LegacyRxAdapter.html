Detailed Description

template<size_t MAX_ELEMENT_SIZE>
class routing::LegacyRxAdapter< MAX_ELEMENT_SIZE >

Extracts the PDUs contained in the messages read from inputReader, adapting the IReader interface, according to the provided RxAdapterTable.

[TPARAMS_BEGIN]
